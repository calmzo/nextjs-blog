## 查看python版本
```bash
which python3
```

```bash
which python3
/Library/Frameworks/Python.framework/Versions/3.10/bin/python3
```

## 安装python
**1.下载**
从官方下载网站获得任何Python版本下载
[https://www.python.org/downloads/release/python-2718/](https://www.python.org/downloads/release/python-2718/)
 运行安装即可
```bash
which python2.7
which python3.10
```

**2. 默认安装目录**
```bash
/Library/Frameworks/Python.framework/Versions/3.10
/Library/Frameworks/Python.framework/Versions/2.7
```

**3. 配置环境变量**
1）终端输入：``vim ~/.bash_profile`` 编辑文件并输入如下内容：
```text
# python 3
#alias python="/Library/Frameworks/Python.framework/Versions/3.10/bin/python3.10"
#export PATH=$PATH:/Library/Frameworks/Python.framework/Versions/3.10/bin/pip3.10
#alias pip="/Library/Frameworks/Python.framework/Versions/3.10/bin/pip3.10"

# python 2
alias python="/Library/Frameworks/Python.framework/Versions/2.7/bin/python2.7"
export PATH=$PATH:/Library/Frameworks/Python.framework/Versions/2.7/bin/pip2.7
alias pip="/Library/Frameworks/Python.framework/Versions/2.7/bin/pip2.7"
```

**4. 查看版本**
```bash
python2 -V
python3 -V 
```
