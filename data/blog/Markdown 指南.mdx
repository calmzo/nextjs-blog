# 什么是 Markdown？

Markdown 是一种在网页上为文本添加样式的方式。你可以控制文档的呈现：将文字加粗或斜体、添加图片、创建列表等。多数情况下，Markdown 只是普通文本外加少量非字母字符，例如 `#` 或 `*`。

# 语法指南

以下概览展示了可在 GitHub.com 或你自己的文本文件中使用的 Markdown 语法。

## 标题

```
# 这是 h1 标题

## 这是 h2 标题

#### 这是 h4 标题
```

# 这是 h1 标题

## 这是 h2 标题

#### 这是 h4 标题

## 强调

```
_这段文本会呈现斜体_

**这段文本会呈现粗体**

_你 **可以** 同时使用_
```

_This text will be italic_

**This text will be bold**

_You **can** combine them_

```
1. 项目 1
1. 项目 2
```

## 脚注

```
这里是一个简单的脚注[^1]。后面还可以跟一些补充文字。

[^1]: 我的参考。
```

Here is a simple footnote[^1]. With some additional text after it.

[^1]: My reference.

- [x] list syntax required (any unordered or ordered list supported)

## 表格

你可以通过罗列单词并用连字符 `-`（用于第一行）划分，再用竖线 `|` 分隔每一列，从而创建表格：

```
| 第一列标题                  | 第二列标题                    |
| --------------------------- | ---------------------------- |
| 单元格 1 的内容             | 单元格 2 的内容               |
| 第一列中的内容              | 第二列中的内容                |
```

| 第一列标题                  | 第二列标题                    |
| --------------------------- | ---------------------------- |
| 单元格 1 的内容             | 单元格 2 的内容               |
| 第一列中的内容              | 第二列中的内容                |

## 删除线

任意被两个波浪号包裹的单词（如 `~~this~~`）都会显示为 ~~删除线~~。
